## Description

If you want to collect feedback and build relationships with your customers you can use WP Feedier plugin for surveys

## Installation

1. Upload the `wp-feedier` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register on https://feedier.com/ and create a  survey.
4. Fill in Public key and Private key, choose survey and other settings.
